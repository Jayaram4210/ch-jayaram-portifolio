# Ch Jayaram Portfolio

This is my personal portfolio website, built using **Lovable**.  
It showcases my resume, LinkedIn profile, and projects in a clean, recruiter‑friendly format.

## 🚀 Features
- Integrated **resume** for quick access
- LinkedIn profile link for professional networking
- Responsive design for desktop and mobile
- Built with Lovable’s AI‑powered project generator
- Easy to extend and customize

## 📂 Project Structure
- `index.html` – Main landing page
- `assets/` – Images, icons, and static resources
- `styles/` – CSS files for layout and design
- `scripts/` – JavaScript functionality
- Additional files generated by Lovable

## 🛠️ Tech Stack
- **Lovable.dev** (project generator)
- **HTML, CSS, JavaScript**
- Optional integrations: GitHub Pages / Vercel for deployment
